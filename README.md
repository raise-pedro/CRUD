Proposta do desafio:
Precisamos organizar o cadastro dos estudantes do FAST e exibir numa lista para
facilitar a visualização dos organizadores.

Vamos criar um programa que simule um sistema de cadastro de estudantes.
O programa deve permitir ao usuário:
- Adicionar novos estudantes, solicitando o nome dos estudantes;
- Listar todos os estudantes cadastrados;
- Buscar um estudante pelo nome;
- Visualizar o menu de opções das ações que ele pode fazer;
- Escolher uma ação, fornecendo uma entrada ao programa um número de opção.

- Ao iniciar, se o usuário selecionar "2. Listar Estudantes", deverá receber uma lista vazia.
- A lista só estará com Estudantes se o usuário adicioná-los através da opção "1. Adicionar Estudante"
- Ao "2. Listar Estudantes", devemos ver o nome do estudante.

- Ao tentar adicionar um estudante com a opção "1. Adicionar Estudante":
- o sistema deverá solicitar ao usuário o nome do estudante.
- o sistema deverá verificar se o nome do estudante já existe antes de adicioná-lo
- Se já existir, o sistema deverá emitir uma mensagem de "Estudante já cadastrado", se não, deverá cadastrar o estudante;
  
- Ao finalizar uma ação, o sistema deverá retornar ao menu inicial;
- Ao selecionar a opção "3. Buscar Estudante", o usuário deverá receber de volta o nome do estudante e uma mensagem avisando se ele foi encontrado ou não.
- Deve-se usar funções para compor o código.

Desafios Extras:
- Permita que o usuário edite os dados de um estudante existente.
- Ao listar o nome dos alunos, listar em ordem alfabética.
- Remover um estudante pelo nome;
- Ao adicionar estudantes, solicitar mais informações sobre ele como idade e cidade onde vive;
